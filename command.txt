1. set username and email
    $ git config --global user.name "Your Name"
    $ git config --global user.email "email@example.com"
2. create a new repository
    $ mkdir <folder name>	创建文件夹
    $ cd <folder name>		进入该文件夹
    $ pwd             		显示文件夹目录
3. make this folder managed by git using comand git init
    $ git init		初始化一个空的仓库，管理版本库
4. add file to repository
    $ git add <file name>
    $ git commit -m "description about commit"
5. looking for something has been changed or not
    $ git status	查询仓库状态
6. looking for what has been changed
    $ git diff <file name>	查看修改内容
